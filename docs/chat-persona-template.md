# Chat Persona Template

Copy this file into a gitignored fixture folder, for example:

```txt
test-data/generated/<persona-slug>/persona.md
```

Then fill it in before creating `chatlog.json`.

## Title

Short label for the persona.

## Age

Age or age range.

## Gender

Gender identity if relevant to the scenario.

## Background

Brief context: life stage, work/school/family situation, location if relevant, and why they are using meCove.

## Writing Style

How this person writes in WhatsApp: short/long, formal/casual, reflective, fragmented, anxious, direct, etc.

## Core Themes

Recurring topics that should appear across the chat log.

## Relationships

Important people who may appear in the logs: family, partner, friends, coworkers, manager, therapist, etc.

## Risk / Safety

Any sensitive content to include or avoid: self-harm, panic, abuse, substance use, violence, medical issues, or none.

## Desired Arc

How the 15-day period should evolve. Include starting point, middle shifts, and ending state.

## Report Window

Default: 15 days.

Mention if the report should cover a different number of days.

## Message Density

Expected rhythm: daily, intermittent, sparse, many messages per day, mostly one long message, etc.

## Notes

Anything else the chat log or report should test.
